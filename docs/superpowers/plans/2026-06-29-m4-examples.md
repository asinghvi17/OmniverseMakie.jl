# M4 Examples Gallery Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Port the 14 portable `references/RPRMakieNotes/` ray-tracing scenes into a self-contained `examples/` gallery rendered end-to-end through OmniverseMakie, gated by a `run_all.jl` property-assert runner.

**Architecture:** A separate `examples/` Pkg environment (devs OmniverseMakie + example deps). Each scene is a small file in `examples/rpr/<name>.jl` defining `scene_<name>() -> Figure` and `assert_<name>(img)`. A shared `common/harness.jl` provides `run_example` (activate → build → `Makie.save` → read-back image) and property-assert helpers. Rendering is subprocess-isolated (ovrtx installs carb signal handlers): `run_all.jl` (parent) spawns `common/run_one.jl <scene>` (child) per scene. Assets live in a gitignored `examples/assets/`, populated once by `fetch_assets.jl` (copy from `references/` + downloads). Renders are committed to `examples/renders/`.

**Tech Stack:** Julia 1.12, OmniverseMakie (dev path), Makie 0.24.12, GeometryBasics, Colors, FileIO/ImageIO, Downloads, Luxor, CSV/DataFrames, GeoMakie/GeoInterface/GeoJSON.

## Porting Protocol (shared — every scene task applies this)

**1. Imports.** Replace the original's `using GLMakie, GeometryBasics` / `using RPRMakie, RadeonProRender` / `using FileIO` / `include("saveRPR.jl")` with `using OmniverseMakie` plus only the extra packages the scene needs (`GeometryBasics`, `Colors`, `FileIO`, `Downloads`→removed, `Luxor`, `CSV`, `DataFrames`, `GeoMakie`, `GeoInterface`, `GeoJSON`). `harness.jl` already does `using OmniverseMakie, ColorTypes, FixedPointNumbers, FileIO`.

**2. Drop the RPR plumbing.** Remove `screen = RPRMakie.RPRScreen(…)`, `matsys = screen.matsys`, `RPRMakie.replace_scene_rpr!(…)`, `colorbuffer(screen)`, `GLMakie.activate!()`, `display(fig)`, every `save(…)` / `saveRPR(…)` line. The harness renders + saves.

**3. Wrap as a builder.** Body becomes `function scene_<name>() … return fig end`. Use `Figure(; size=(W,H))` (the original's `resolution=` → `size=`). Build the LScene exactly as the original: `ax = LScene(fig[1,1]; show_axis=false, scenekw=(;lights=lights))`. Keep the camera lines (`zoom!(ax.scene, cameracontrols(ax.scene), f)`, and any `update_cam!(ax.scene, eye, lookat, up)` / `cam = cameracontrols(ax.scene); cam.eyeposition[]=…; update_cam!(ax.scene, cam)`).

**4. Translate materials** (drop `matsys` arg, replace the `material=` value):

| Original | Port |
|---|---|
| `material = RPR.DiffuseMaterial(matsys)` | **drop `material=`** entirely; keep `color=` (USD displayColor matte) |
| `material = RPR.EmissiveMaterial(matsys; …)` with colour `c` | `material = (; emissive = c)` |
| `material = RPR.Glass(matsys)` | `material = (; opacity = 0.15f0, roughness = 0.0f0, metallic = 0.0f0)` |
| `RPR.UberMaterial(…; reflection_metalness=m, reflection_roughness=r, …)` | `material = (; metallic = Float32(m), roughness = Float32(r))` |
| `RPR.UberMaterial(…; transparency=t, …)` | `material = (; opacity = Float32(1 - t))` |
| `color = img'` (an image/texture) | keep `color = img'` (backend auto-emits `diffuse_texture` + `st`) |

`material=` friendly keys (verbatim): `:metallic`, `:roughness`, `:opacity`, `:base_color`, `:emissive`, `:base_color_texture`, `:normal_texture`, `:roughness_texture`, `:metallic_texture`. All scalar values `Float32`; colours any `Colorant`.

**5. Assets.** Replace `load(Downloads.download(url))` and `load("./imgs/x.png")` with `FileIO.load(asset("<scene>", "<dest>"))` using the dest names from the Task 2 manifest (e.g. `asset("uberMExample", "earth.jpg")`). NO `Downloads.download` at render time.

**6. Lights.** Keep `lights = [EnvironmentLight(1.0, img'), PointLight(…)]`. For a real EXR/JPEG env map the texture is not honored (best-effort dome) — pass a neutral 1×1 colorant instead and add a docstring note (only `reflections_glass_material` is affected).

**7. Asserts.** Add `function assert_<name>(img) … end` using harness helpers (`assert_nonblack`, `nonblack_count`, `color_fraction`, `lum`). Each assert: at minimum `assert_nonblack(img, "<name>")`, plus one scene-appropriate signal (a dominant colour, a bright-specular fraction, etc.). `@assert` messages start with `"FAIL: <name>"`.

**Scene-file skeleton (every `examples/rpr/<name>.jl`):**

```julia
# examples/common/harness.jl — shared render harness + property-assert helpers.
# Loaded by common/run_one.jl inside each render subprocess (and by harness_test.jl).
using OmniverseMakie
using ColorTypes, FixedPointNumbers, FileIO

const COMMON_DIR  = @__DIR__
const EXAMPLES_DIR = normpath(joinpath(COMMON_DIR, ".."))
const RENDERS_DIR  = joinpath(EXAMPLES_DIR, "renders")
const ASSETS_DIR   = joinpath(EXAMPLES_DIR, "assets")

"""
    asset(scene, relpath) -> String

Absolute path to `examples/assets/<scene>/<relpath>`; errors (telling the user to run
`fetch_assets.jl`) if missing.
"""
function asset(scene::AbstractString, relpath::AbstractString)
    p = joinpath(ASSETS_DIR, scene, relpath)
    isfile(p) || error("OmniverseMakie examples: asset '$(scene)/$(relpath)' not found at \
                        $p — run `julia --project=examples examples/fetch_assets.jl` first.")
    return p
end

"""
    run_example(name, scene_fn; size, out) -> Matrix

Activate OmniverseMakie, build the figure via `scene_fn()`, `Makie.save` it to `out`
(routes through our colorbuffer), and return the saved image read back for asserts.
"""
function run_example(name::AbstractString, scene_fn;
                     size = (900, 900),
                     out  = joinpath(RENDERS_DIR, name * ".png"))
    OmniverseMakie.activate!()
    fig = scene_fn()
    mkpath(dirname(out))
    Makie.save(out, fig)
    return FileIO.load(out)
end

# --- property-assert helpers (M1–M3 idiom) -----------------------------------
lum(c) = 0.2126f0 * Float32(red(c)) + 0.7152f0 * Float32(green(c)) + 0.0722f0 * Float32(blue(c))

nonblack_count(img; thresh = 0.02f0) = count(c -> lum(c) > thresh, img)

function assert_nonblack(img, name; frac = 0.01)
    n = nonblack_count(img); total = length(img)
    @assert n > frac * total "FAIL: $(name) render is (near) black: $(n)/$(total) nonblack \
                              (< $(round(frac*100; digits=2))%)"
    return n
end

"""
    color_fraction(img, which; thresh) -> Float64

Fraction of pixels where channel `which` (:red/:green/:blue) is the max channel and the
pixel is bright enough (max channel > thresh). Robust colour-presence signal under RTX noise.
"""
function color_fraction(img, which::Symbol; thresh = 0.15f0)
    n = count(img) do c
        r, g, b = Float32(red(c)), Float32(green(c)), Float32(blue(c))
        m = max(r, g, b)
        m <= thresh && return false
        which === :red   ? (r >= g && r >= b) :
        which === :green ? (g >= r && g >= b) :
                           (b >= r && b >= g)
    end
    return n / length(img)
end
```

```julia
function fetch_one(scene, dest, kind, src)
    destdir = joinpath(ASSETS_DIR, scene); mkpath(destdir)
    destpath = joinpath(destdir, dest)
    if isfile(destpath); println("  have     $(scene)/$(dest)"); return destpath; end
    if kind === :copy
        s = joinpath(RPRN, src); isfile(s) || error("missing reference asset: $(s)")
        cp(s, destpath; force = true); println("  copied   $(scene)/$(dest)")
    else
        println("  download $(scene)/$(dest) …"); Downloads.download(src, destpath)
    end
    return destpath
end

let want = isempty(ARGS) ? MANIFEST : filter(e -> e[1] in ARGS, MANIFEST)
    for (scene, dest, kind, src) in want
        fetch_one(scene, dest, kind, src)
    end
end
```

```julia
# Ported from references/RPRMakieNotes/scripts/sphere_plane_greysky.jl (Lazaro Alonso).
# Blue diffuse sphere on a gainsboro plane under a grey dome + warm point light.
# DiffuseMaterial → plain color= (USD displayColor matte).
using OmniverseMakie, GeometryBasics, Colors

SphereTess(; o = Point3f(0), r = 1, tess = 64) = uv_normal_mesh(Tesselation(Sphere(o, r), tess))

function scene_sphere_plane_greysky()
    grey   = [colorant"grey90" for _ in 1:1, _ in 1:1]
    lights = [EnvironmentLight(1.0, grey'), PointLight(Vec3f(2, 0, 2.0), RGBf(8.0, 6.0, 5.0))]
    plane  = Rect3f(Vec3f(-5, -2, -1.05), Vec3f(10, 4, 0.05))
    fig = Figure(; size = (900, 900))
    ax  = LScene(fig[1, 1]; show_axis = false, scenekw = (; lights = lights))
    mesh!(ax, SphereTess(); color = RGB(0.082, 0.643, 0.918))
    mesh!(ax, plane; color = :gainsboro)
    zoom!(ax.scene, cameracontrols(ax.scene), 0.22)
    return fig
end

function assert_sphere_plane_greysky(img)
    assert_nonblack(img, "sphere_plane_greysky"; frac = 0.05)
    bf = color_fraction(img, :blue)
    @assert bf > 0.01 "FAIL: sphere_plane_greysky expected a blue sphere; blue_fraction=$(bf)"
    println("  blue_fraction=", bf)
end
```

```julia
# examples/run_all.jl — the M4 gate. Subprocess-render every scene + property-assert.
# Usage: julia --project=examples examples/run_all.jl            # all rpr/*.jl
#        julia --project=examples examples/run_all.jl <scene…>   # subset
const EXAMPLES_DIR = @__DIR__
const RPR_DIR  = joinpath(EXAMPLES_DIR, "rpr")
const RUN_ONE  = joinpath(EXAMPLES_DIR, "common", "run_one.jl")
const OVRTX_LIB = get(ENV, "OVRTX_LIBRARY_PATH",
    "/home/juliahub/temp/omniverse-makie/references/ovrtx/examples/python/minimal/.venv/lib/python3.13/site-packages/ovrtx/bin/libovrtx-dynamic.so")

scenes = isempty(ARGS) ?
    sort([replace(basename(f), ".jl" => "") for f in readdir(RPR_DIR; join = true) if endswith(f, ".jl")]) :
    ARGS

results = Tuple{String,Symbol}[]
for s in scenes
    println("\n=== rendering $(s) ===")
    cmd = setenv(`julia --project=$(EXAMPLES_DIR) $(RUN_ONE) $(s)`,
                 "OVRTX_LIBRARY_PATH" => OVRTX_LIB,
                 "PATH" => get(ENV, "PATH", ""), "HOME" => get(ENV, "HOME", ""))
    ok = success(pipeline(cmd; stdout = stdout, stderr = stderr))
    push!(results, (s, ok ? :pass : :fail))
end
```
